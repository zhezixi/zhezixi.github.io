The 10000 Floors Elevator Challenge 

Project Goal

Design alternative control interfaces for a 10000 floors elevator.

Project context

The way I conquered this challenge is to design a conrol interface which can operate in NFC supported portable devices like a cellphone.

Involvement

UI/UX design

Date

Feb 2019

Tools

Adobe XD

Individual project

Inspired by Introduction to User Experience Principles and Processes course provided by University of Michigan

Criteria/constraints I discovered

Problem 1 : The capacity of this elevator is much bigger than normal elevator. It might be hard to input the floor one by one each time.
Solutions:
1)Personal/wearable devices
easier for users enter the floors they want to achieve at the same time.
2)Multiple panels inside the elevator
-Friendly and simple interface
-Straightforward input and control.
3)Users finish their floor option input when they are waiting for the elevator Floor choice input outside the elevator.
-NFC/RFID for quicker input-apartment/office building choices.
Problem 2 : There are too many floors, difficulties when choosing the floor.
Solutions:
1)Name search function-works for offices building(apartment needs more privacy)
2)Authorities control:
-ID card scan/QR code scan-shows floorsâ€™ informations users usually achieve, and the floors they have authorities.
